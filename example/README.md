# نمونه Fetchy SDK Flutter

این اپ یک **نمونهٔ آماده** است تا ببینی `fetchy_sdk_flutter` چطور کار می‌کند.

برای راهنمای کامل نصب در پروژهٔ خودت، [`README.md`](../README.md) اصلی را بخوان.

---

## قبل از اجرا

- Flutter روی سیستم نصب باشد
- یک دستگاه یا emulator اندروید داشته باشی
- اکانت در [پنل Fetchy](https://fetchy.notifhub.com) و فایل تنظیمات اپ

---

## مراحل اجرا

### ۱ — گذاشتن فایل تنظیمات

1. برو به [پنل Fetchy](https://fetchy.notifhub.com) و اپت را ثبت کن.
2. فایل JSON تنظیمات را دانلود کن.
3. اسم فایل را عوض کن به: `fetchy-config.json`
4. فایل را اینجا بگذار (اگر پوشهٔ `assets` نبود، بسازش):

```txt
android/app/src/main/assets/fetchy-config.json
```

می‌توانی از [`fetchy-config.sample.json`](fetchy-config.sample.json) ساختار فایل را ببینی، ولی برای اجرای واقعی باید فایل خودت را از پنل بگیری.

> این فایل کلید API دارد. داخل Git commit نکن.

### ۲ — اجرای اپ

از همین پوشه (`example/`) در ترمینال بزن:

```bash
flutter pub get
flutter run
```

تنظیمات Gradle و AndroidManifest در این نمونه از قبل انجام شده؛ نیازی به تغییرشان نیست.

---

## این نمونه چه چیزهایی را نشان می‌دهد؟

کد اصلی در [`lib/main.dart`](lib/main.dart) است:

- راه‌اندازی SDK هنگام باز شدن اپ
- گرفتن اجازهٔ نوتیفیکیشن از کاربر
- نمایش **توکن دستگاه** (با دکمهٔ کپی)
- نمایش وضعیت اجازهٔ نوتیفیکیشن
- بروزرسانی و همگام‌سازی وضعیت اجازه با سرور Fetchy

---

## اگر گیر کردی

| مشکل | راه‌حل |
|------|--------|
| خطای `:fetchy-sdk` در build | `flutter pub get` بزن، بعد دوباره `flutter run` |
| توکن نمی‌آید | چند ثانیه صبر کن؛ اجازهٔ نوتیف و `fetchy-config.json` را چک کن |
| اپ crash می‌کند | مطمئن شو `fetchy-config.json` در مسیر `assets` هست و API key درست است |
